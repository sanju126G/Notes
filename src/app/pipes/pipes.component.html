Pipes In Angular

  • The Angular Pipe takes the raw data as input and then transforms the raw data into some desired format.
  • There Are four Types of pipes that we can use in angular :-

    1. Built-in pipes
      1. lowercase.

        • {{uppercase | lowercase}}

      2. uppercase.

        • {{lowercase | uppercase}}

      3. currency.

        • before using pipe {{currency}} | after using pipe {{currency | currency}}

      4. date.

        • before using pipe {{date}} | after using pipe {{date | date}}

      5. percent.

        • before using pipe {{percentage}} | after using pipe {{percentage | percent}}


    2. Parameterized Pipes

      • we can pass one are more parameters to pipe.

        before using pipe {{date}}

        • after using pipe {{date | date : 'short'}}

        • after using pipe {{date | date : 'long'}}

        • after using pipe {{date | date : 'medium'}}

    3. Chaining pipes.

      • we can connect multiple pipes to a single data input.

    4. Custom pipes.

      • In addtion to built-in pipes we can also create our own pipes using the Command ng g p pipeName
